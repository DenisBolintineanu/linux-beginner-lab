# 🐧 linux-learning-lab

A hands-on Linux training parcours for **absolute beginners**.
Learn the Linux command line by solving small, practical challenges directly in your terminal.

---

## 🎯 What is this?

**linux-learning-lab** is a beginner-friendly learning project designed to teach you the basics of Linux step by step.

Instead of reading theory, you will:

* explore directories
* search for files
* fix problems
* solve small "missions"

Think of it as a **mini game for the terminal**.

---

## 👶 Who is this for?

This project is for you if:

* you have little to no Linux experience
* you want to learn the command line (CLI)
* you prefer **learning by doing**
* you are tired of dry tutorials

---

## 🚀 How it works

Each level is a small challenge with a clear goal.

Example:

* Find a hidden file
* Fix a broken directory structure
* Search for a specific string
* Solve permission issues

Every level includes:

* a short explanation
* a task
* optional hints
* a solution (if you get stuck)

---

## 📦 Setup

Clone the repository:

```bash
git clone https://github.com/your-username/linux-learning-lab.git
cd linux-learning-lab
```

(Optional – if you add a setup script later)

```bash
bash setup.sh
```

---

## 🧩 Levels

```text
01-navigation-basics     → ls, cd, cat
02-paths                → working with paths (ls, cat without cd)
03-files                → mkdir, touch, cp, mv, rm
04-search               → find, grep
05-permissions          → chmod, chown
06-pipes                → |, sort, uniq, wc
07-archives             → tar, gzip
08-processes            → ps, top, kill
```

Start with:

```bash
cd 01-navigation-basics
```

## 🕹 Example Challenge

> Somewhere in a maze of directories, a file contains a secret code.

Commands you might need:

* `ls`
* `cd`

## 💡 Rules

* Use only the terminal
* No GUI file explorer
* Try to solve it yourself before looking at hints
