# 🧩 Level 01 – Navigation Basics

Welcome to your first challenge 👋

In this level, you will learn how to **navigate through directories** and **read files** using the Linux terminal.

---

## 🎯 Learning Goals

After this level, you should understand:

* how to list files with `ls`
* how to move between directories with `cd`
* how to read files with `cat`

---

## 🕹 Your Mission

Somewhere inside this directory, a **secret file** is hidden.

Your goal is to:

> 🔍 **Find the file and read its content**

At the end, you should see a **secret code**.

---

## 📁 Where to start

You are currently in:

```bash
01-navigation-basics
```

Start exploring from here.

Use `cd maze` to move into the directory called "maze".

---

## 🧠 Hints

Use `ls` to see what is inside a directory.

Use `cd foldername` to move into a directory.

Use cd .. to go back if you reached a dead end.

Use `cat filename` to read a file.

Not every path is correct 👀
Some files might lead you in the wrong direction.

---

## ⚠️ Rules

* Do **not** use a graphical file explorer
* Use only terminal commands
* Try it yourself before opening all hints

---

## ✅ Success

You completed the level when you find something like:

```text
LEVEL 1 COMPLETE
Code: _______
```

---

## 🆘 Solution

There are multiple ways to solve this.

Basic approach:

1. Use `ls` to explore directories
2. Use `cd` to go deeper
3. Use `cat` to read files

Eventually, you will find the correct path leading to the file with the code.
